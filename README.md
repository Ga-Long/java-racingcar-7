# [자동차 경주] 기능 목록

## 입력
* 자동차 이름 입력
* 횟수 입력


## 출력
* 차의 전진 결과 출력
* 우승자 한 명인 경우 출력
* 우승자 여러 명인 경우 쉼표로 구분하여 출력


## 자동차 경주 게임
* 자동차 이름은 쉼표(,)를 기준으로 구분
* 0~9 사이 무작위 값을 만듦
* 무작위 값이 4 이상인 경우 전진
* 횟수만큼 반복
* 많이 전진한 자동차 우승자 결정


## 예외처리
**이름** 
* 이름 5자 초과 예외처리
* 이름 입력하지 않으면 예외처리
* 이름 중복 예외처리

**횟수**
* 양수가 아닌 경우 예외처리
